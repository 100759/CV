个人简历网站 - 启动说明
========================

项目已成功创建！如果遇到终端编码问题，请按以下步骤操作：

方法一：使用命令提示符（推荐）
1. 按 Win + R，输入 cmd，回车
2. 在命令提示符中，切换到项目目录：
   cd "C:\Users\Administrator\Desktop\cn简历"
3. 安装依赖：
   npm install
4. 启动开发服务器：
   npm run dev

方法二：使用PowerShell
1. 右键点击项目文件夹
2. 选择"在终端中打开"
3. 运行：npm install
4. 运行：npm run dev

方法三：使用VS Code
1. 用VS Code打开项目文件夹
2. 按 Ctrl + ` 打开终端
3. 运行：npm install
4. 运行：npm run dev

启动成功后：
- 浏览器会自动打开 http://localhost:3000
- 如果没有自动打开，请手动访问该地址

项目特色：
✅ 现代化响应式设计
✅ 左侧固定导航栏（桌面端）
✅ 移动端汉堡菜单
✅ 流畅的动画效果
✅ 专业的视觉设计

如果遇到任何问题，请检查：
1. 是否已安装 Node.js（版本 16+）
2. 网络连接是否正常
3. 端口 3000 是否被占用

祝您使用愉快！
